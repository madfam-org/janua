# Installation Guide

Complete installation instructions for integrating Janua into your project.

## Prerequisites

Before installing Janua, ensure you have:

- **Node.js 16.8+** for JavaScript/TypeScript projects
- **Python 3.8+** for Python projects
- **Go 1.19+** for Go projects
- A Janua account ([sign up here](https://app.janua.dev))

## JavaScript/TypeScript Installation

### Next.js (Recommended)

For Next.js applications with App Router:

```bash
npm install @janua/nextjs
```

### React

For standalone React applications:

```bash
npm install @janua/react-sdk
```

### Vue

For Vue 3 applications:

```bash
npm install @janua/vue
```

### Universal JavaScript

For other JavaScript frameworks or Node.js:

```bash
npm install @janua/js
```

## Python Installation

### Using pip

```bash
pip install janua-python
```

### Using poetry

```bash
poetry add janua-python
```

### Framework-specific packages

For Django:
```bash
pip install janua-django
```

For FastAPI:
```bash
pip install janua-fastapi
```

For Flask:
```bash
pip install janua-flask
```

```bash
go get github.com/madfam-io/janua-go
```

## Environment Setup

### 1. Create Your Application

1. Go to [app.janua.dev](https://app.janua.dev)
2. Sign up or log in to your account
3. Click "Create New Application"
4. Configure your application settings:
   - **Name**: Your application name
   - **Domain**: Your application domain (e.g., `localhost:3000` for development)
   - **Redirect URIs**: Where users return after authentication
   - **Logout URIs**: Where users go after logout

### 2. Get Your Credentials

After creating your application, you'll receive:

```bash
JANUA_ISSUER=https://api.janua.dev
JANUA_AUDIENCE=your-unique-audience-id
JANUA_CLIENT_ID=your-client-id
JANUA_CLIENT_SECRET=your-client-secret
```

### 3. Environment Variables

Create appropriate environment files for your framework:

#### Next.js (`.env.local`)

```bash
JANUA_ISSUER=https://api.janua.dev
JANUA_AUDIENCE=your-audience
JANUA_CLIENT_ID=your-client-id
JANUA_CLIENT_SECRET=your-client-secret
JANUA_REDIRECT_URI=http://localhost:3000/api/auth/callback
```

#### Python (`.env`)

```bash
JANUA_ISSUER=https://api.janua.dev
JANUA_AUDIENCE=your-audience
JANUA_CLIENT_ID=your-client-id
JANUA_CLIENT_SECRET=your-client-secret
JANUA_REDIRECT_URI=http://localhost:8000/auth/callback
```

#### Go (environment or config file)

```bash
export JANUA_ISSUER=https://api.janua.dev
export JANUA_AUDIENCE=your-audience
export JANUA_CLIENT_ID=your-client-id
export JANUA_CLIENT_SECRET=your-client-secret
```

## Framework-Specific Configuration

### Next.js Configuration

Create `lib/janua.ts`:

```typescript
import \{ Janua \} from '@janua/nextjs'

export const janua = new Janua(\{
  issuer: process.env.JANUA_ISSUER!,
  audience: process.env.JANUA_AUDIENCE!,
  clientId: process.env.JANUA_CLIENT_ID!,
  clientSecret: process.env.JANUA_CLIENT_SECRET!,
  redirectUri: process.env.JANUA_REDIRECT_URI!,
  
  // Optional configuration
  sessionCookieName: 'janua-session',
  sessionDuration: 24 * 60 * 60 * 1000, // 24 hours
  refreshTokenRotation: true,
\})
```

### React Configuration

Create `lib/janua.ts`:

```typescript
import \{ JanuaProvider \} from '@janua/react-sdk'

const januaConfig = \{
  issuer: process.env.REACT_APP_JANUA_ISSUER!,
  audience: process.env.REACT_APP_JANUA_AUDIENCE!,
  clientId: process.env.REACT_APP_JANUA_CLIENT_ID!,
  redirectUri: window.location.origin + '/callback',
\}

// Wrap your app with JanuaProvider
function App() \{
  return (
    <JanuaProvider config=\{januaConfig\}>
      <YourAppComponents />
    </JanuaProvider>
  )
\}
```

### Python Configuration

Create `config/janua.py`:

```python
import os
from janua import Janua

janua = Janua(
    issuer=os.getenv('JANUA_ISSUER'),
    audience=os.getenv('JANUA_AUDIENCE'),
    client_id=os.getenv('JANUA_CLIENT_ID'),
    client_secret=os.getenv('JANUA_CLIENT_SECRET'),
    redirect_uri=os.getenv('JANUA_REDIRECT_URI'),
    
    # Optional configuration
    session_duration=86400,  # 24 hours in seconds
    refresh_token_rotation=True,
)
```

### Go Configuration

Create `config/janua.go`:

```go
package config

import (
    "os"
    "github.com/madfam-io/janua-go"
)

func NewJanuaClient() *janua.Client \{
    return janua.New(\{
        Issuer:       os.Getenv("JANUA_ISSUER"),
        Audience:     os.Getenv("JANUA_AUDIENCE"),
        ClientID:     os.Getenv("JANUA_CLIENT_ID"),
        ClientSecret: os.Getenv("JANUA_CLIENT_SECRET"),
        RedirectURI:  os.Getenv("JANUA_REDIRECT_URI"),
        
        // Optional configuration
        SessionDuration: time.Hour * 24,
        RefreshTokenRotation: true,
    \})
\}
```

## Verification

### Test Your Installation

Create a simple test to verify your installation:

#### JavaScript/TypeScript

```typescript
import \{ janua \} from './lib/janua'

async function testConnection() \{
  try \{
    const config = await janua.getConfiguration()
    console.log('✅ Janua connected successfully:', config.issuer)
  \} catch (error) \{
    console.error('❌ Connection failed:', error)
  \}
\}

testConnection()
```

#### Python

```python
from config.janua import janua

def test_connection():
    try:
        config = janua.get_configuration()
        print(f"✅ Janua connected successfully: \{config['issuer']\}")
    except Exception as error:
        print(f"❌ Connection failed: \{error\}")

if __name__ == "__main__":
    test_connection()
```

#### Go

```go
package main

import (
    "fmt"
    "log"
    "your-app/config"
)

func main() \{
    client := config.NewJanuaClient()
    
    config, err := client.GetConfiguration()
    if err != nil \{
        log.Fatalf("❌ Connection failed: %v", err)
    \}
    
    fmt.Printf("✅ Janua connected successfully: %s\\n", config.Issuer)
\}
```

## Troubleshooting

### Common Issues

#### "Invalid client credentials"
- Verify your `JANUA_CLIENT_ID` and `JANUA_CLIENT_SECRET`
- Ensure no extra spaces in your environment variables
- Check that your application is active in the Janua dashboard

#### "Invalid redirect URI"
- Ensure your redirect URI exactly matches what's configured in Janua
- Include the full path (e.g., `http://localhost:3000/api/auth/callback`)
- For production, use HTTPS URLs

#### "Network connection failed"
- Check your internet connection
- Verify the `JANUA_ISSUER` URL is correct
- Ensure no firewall is blocking requests to `api.janua.dev`

#### "Module not found" errors
- Run `npm install` / `pip install` / `go mod tidy` again
- Clear your package manager cache
- Check that you're using the correct package name for your framework
